# 🔐 助记词分片工具 (Seed Split Tool)

一个基于 Shamir 秘密分享算法的助记词分片工具，可以安全地将助记词分割成多个分片，任意指定数量的分片即可恢复原始助记词。

## ✨ 核心特性

### 🔒 **安全性优先**

- **完全离线运行**：数据不会离开您的设备
- **专业级算法**：使用经过审计的 `shamir-secret-sharing` 库
- **零依赖风险**：构建后的单文件可独立运行
- **断网推荐**：建议在断网环境中使用以获得最高安全性

### 🧩 **灵活的分片配置**

- **助记词支持**：支持 12 或 24 个单词的助记词
- **分片数量**：可生成 3-7 个分片
- **恢复阈值**：可设置 2-5 个分片的恢复门槛
- **任意组合**：任意达到阈值数量的分片都能恢复原始助记词

### 🎨 **优秀的用户体验**

- **现代化界面**：响应式设计，支持各种设备
- **实时验证**：输入分片时自动检测格式和有效性
- **智能提示**：彩色状态指示器和详细的操作指导
- **容错处理**：自动跳过无效分片，智能处理用户输入

## 📖 使用指南

### 1. 生成分片

1. **配置参数**：

   - 选择助记词数量（12 或 24 个单词）
   - 设置分片总数（3-7 个）
   - 设置恢复所需分片数（2-5 个）

2. **输入助记词**：

   - 在对应的输入框中输入内容，系统会自动联想匹配的 BIP39 单词
   - 失去焦点时自动验证，只接受有效的 BIP39 单词
   - 无效输入将被自动清空，请从建议列表中选择单词
   - 绿色边框表示有效单词，红色边框表示无效输入
   - 系统会实时验证输入完整性和有效性

3. **生成分片**：

   - 点击"🔒 生成分片"按钮
   - 系统将生成指定数量的分片
   - 每个分片都包含恢复信息

4. **保存分片**：
   - 点击每个分片旁的"复制"按钮
   - 将分片分别保存在不同的安全位置

### 2. 恢复助记词

1. **准备分片**：

   - 收集至少达到阈值数量的分片
   - 确保分片格式完整且有效

2. **输入分片**：

   - 将分片粘贴到恢复区域的文本框中
   - 每行一个分片，系统会自动处理换行
   - 实时状态指示器会显示分片验证结果

3. **恢复助记词**：
   - 当分片数量足够时，"恢复助记词"按钮会自动启用
   - 点击按钮即可恢复原始助记词

### 核心算法

项目使用 **Shamir 秘密分享算法**，这是一种基于多项式插值的密码学算法：

- **数学基础**：基于拉格朗日插值和有限域运算
- **安全保证**：任何少于阈值数量的分片都无法获得原始信息
- **完美安全**：在信息论意义上是完全安全的
- **灵活性**：支持 (k, n) 门限方案，其中 k 为阈值，n 为总分片数

## 🔍 常见问题

### Q: 分片丢失了怎么办？

A: 只要剩余分片数量达到设定的阈值，就可以恢复原始助记词。这就是 Shamir 算法的优势。

### Q: 可以修改已生成的分片吗？

A: 不可以。分片是通过数学算法生成的，任何修改都会导致无法恢复原始数据。

### Q: 支持哪些助记词标准？

A: 工具严格支持 BIP39 标准助记词，所有输入的单词必须是有效的 BIP39 单词。系统会自动验证输入内容，无效单词将被拒绝。

### Q: 分片可以在不同设备间使用吗？

A: 可以。分片是标准的 Base64 编码字符串，可以在任何支持该工具的设备上使用。
